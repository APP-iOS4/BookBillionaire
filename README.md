## ADS
책을 사랑하는 사용자들이 구매 부담 없이 다른 사용자들과 책을 대여하면서 책을 소유하지 않고도 다양한 책을 즐길 수 있는 앱
<br><br>

### 구현 목표
- 책 대여를 제공하는 사람과 책을 대여하는 사람사이의 원활한 과정 중개
- 책의 분류와 해당 책에 대한 설명확인 후 대여 진행
- 보유하고 있는 책의 업로드 및 해당 책의 대여 가능 여부 선택기능
- 채팅기능을 통한 1:1 메세지전송
<br>
